# custom-openwrt-firmware

個人的に必要なOpenWrtのカスタムファームウェアをビルドするプロジェクトです。

現在下記のイメージがあります。

 * OpenWrt
   * 15.05.1
 * デバイス
   * Buffalo WZR-HP-AG300H / WZR-600DHP
   * Buffalo WZR-HP-G450H
 * カスタマイズ内容
   * kernel FPU emulation有効
